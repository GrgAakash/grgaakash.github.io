Introduction

In Part I, for the n=2 case, we relied on a “happy accident of geometry.” But as we step into dimensions greater than 2, this geometric safety net vanishes. For instance, we saw in the 3D comparison from Part I that the chopping argument fails because the energy integral converges even for singular functions without violating the global energy bounds. Furthermore, the “algebraic slack” in the linearized equation from Part I allows the gradient map to stretch space infinitely in one direction while remaining valid.

We are thus left in a position where the geometry is too flexible to force regularity. To solve Hilbert’s problem in higher dimensions, we must abandon the “shape” of the gradient and instead look at the “mass” of the gradient. We need to stop asking “what does it look like?” and start asking “how much energy does it carry?”.

This requires a new set of tools. Before we can appreciate the final solution, we must establish the prerequisites of some analytical toolkit: Weak Derivatives, Sobolev Spaces, the Sobolev Inequality, and the Caccioppoli inequality that already appeared in Part I.

Weak Derivatives

Before going into what weak derivatives are, we need two ideas that extend what we learned in our introductory real analysis or honors calculus class:

Definition 1 (Locally Integrable)

When we work with weak derivatives, we don’t need the whole function to be integrable over an open subset, Ω⊆Rn. It’s enough that for all compact interval K, the integral ∫K​∣u(x)∣dx exists. These functions are called locally integrable functions. We represent the set of such functions by Lloc1​(Ω).

L1 just means the absolute value of the function has a finite integral.

Definition 2 (Test Functions)

A function ϕ:Rn→R is a test function if:

  1. Smoothness: ϕ is infinitely differentiable, i.e., ϕ∈C∞(Rn).

  2. *Compact Support: There exists a compact set K⊂Rn such that

    ϕ(x)=0,      ∀x∈/K.

Note that in Rn, “compact” means the set is closed and bounded. We write the space of test functions as Cc∞​(Ω), where Ω is an open subset of Rn.

Before moving on, let us work with u(x)=∣x∣ and suppose for some g(x) we want to find

∫−∞∞​(∣x∣ g(x))′dx.

Now, we have

∫−∞∞​(∣x∣ g(x))′dx=∫−∞∞​(∣x∣)′g(x)dx+∫−∞∞​∣x∣g(x)′dx.

The middle term seem to be problematic due to the presence of (∣x∣)′, and it definitely is. Does that mean problems of this kind are unsolvable? Fortunately, we are not completely out of luck; if g(x) is any test function ϕ∈Cc∞​(R), we immediately observe that

[∣x∣ϕ(x)]−∞∞​=∫−∞∞​(∣x∣)′ϕdx+∫−∞∞​∣x∣ϕ′dx.

∫−∞∞​(∣x∣)′ϕ(x)dx=−∫−∞∞​∣x∣ϕ′(x)dx

Now, let us try solving the right-hand side where our burden of having well-defined derivatives/smoothness transfers from ∣x∣ to our test function ϕ(x), which, by definition, is smooth.

The right-hand side of → can be written as

=−(∫−∞0​(−x)ϕ′(x)dx+∫0∞​xϕ′(x)dx).

For x<0,

∫−∞0​(−x)ϕ′(x)dx==0[−xϕ(x)]−∞0​​​−∫−∞0​(−1)ϕ(x)dx=∫−∞0​ϕ(x)dx.

Similarly, for x>0,

∫0∞​xϕ′(x)dx==0[xϕ(x)]0∞​​​−∫0∞​1⋅ϕ(x)dx=−∫0∞​ϕ(x)dx.

Therefore,

∫−∞0​ϕ(x)dx−∫0∞​ϕ(x)dx=−(∫−∞0​(−1)ϕ(x)dx+∫0∞​(1)ϕ(x)dx)=−∫−∞∞​sgn(x)ϕ(x)dx.

The single point x=0 contributes nothing to the integral, so we don’t need to define sgn(0). Summarizing,

∫−∞∞​(∣x∣)′ϕ(x)dx=−∫−∞∞​∣x∣ϕ′(x)dx=∫−∞∞​sgn(x)ϕ(x)dx.

Just looking at the left-most and right-most integrals we can see something interesting; it looks like (∣x∣)′ is equal to sgn(x). Well, it is, but in a weak sense. As we already know that (∣x∣)′ does not exist everywhere so we cannot really say that (∣x∣)′=sgn(x),  ∀x∈R. So, what do we do? We name sgn(x) as the weak derivative. These so-called Weak Derivatives are introduced so that there is a way to talk about “slope” as in our above example without breaking the rules of calculus. In physics and in variational calculus, we often deal with functions similar to ∣x∣ and the idea here is to shift the burden of smoothness from the function u (which might be rough) onto a “test function” ϕ. This motivates our definition for weak derivatives:

Definition 3 (Weak Derivative in R)

Let u be a locally integrable function. We say a function v is the weak derivative of u if, for every test function ϕ∈Cc∞​(Ω) where Ω⊆R, the following equality holds:

∫Ω​u(x)ϕ′(x)dx=−∫Ω​v(x)ϕ(x)dx.

If such a v exists, it is essentially unique, and we write u′=v in the weak sense.

Definition 4 (Weak Derivative in Rn)

Let u∈Lloc1​(Ω), where Ω⊆Rn is an open set and α be a multi index2. We say a locally integrable function v∈Lloc1​(Ω) is the weak α-derivative of u if, for every test function ϕ∈Cc∞​(Ω), the following holds:

∫Ω​u(x)Dαϕ(x)dx=(−1)∣α∣∫Ω​v(x)ϕ(x)dx.

If such a v exists, it is essentially unique, and we write Dαu=v in the weak sense.

Now, let us do a specific example: Let u(x1​,x2​)=∣x1​∣x2​ in R2. Let us find its weak partial derivative with respect to x1​ which is α=(1,0), so Dαϕ=∂x1​∂ϕ​ and ∣α∣=1.

Now by using Fubini’s Theorem3, we have

∫R2​u(x)Dαϕ(x)dx=∫−∞∞​x2​(∫−∞∞​∣x1​∣∂x1​∂ϕ​dx1​)dx2​.

The inner integral is exactly the 1D case we already did. Thus

∫−∞∞​x2​(−∫−∞∞​sgn(x1​)ϕ(x1​,x2​)dx1​)dx2​=−∫R2​(sgn(x1​)x2​)ϕ(x)dx1​dx2​.

By comparing this to the right-hand side of our definition, (−1)∣α∣∫vϕdx, we immediately see that the weak derivative is v(x1​,x2​)=sgn(x1​)x2​. In our above examples, both ∣x1​∣x2​ and ∣x∣ have a weak derivative as it fails to be classically differentiable at a single point for ∣x∣, and the line x1​=0 for ∣x1​∣x2​. Sets like these, where classical differentiation is undefined, have measure zero4, so the integral isn’t affected and thus our definition stays intact. You might think that if a function is differentiable everywhere apart from a set of measure zero, then its classical derivative (where defined) is automatically its weak derivative. However, this is not the case, a counterexample is the Cantor function f:[0,1]→[0,1], which is continuous, non-decreasing, and differentiable almost everywhere with f′=0 a.e. Yet, it does not have a weak derivative in Lloc1​. If f′=0 were the weak derivative, then ∫01​fϕ′dx would have to equal 0 for every test function ϕ, but this is false since f increases from 0 to 1. Overall the concept of weak derivatives allows us to do calculus on functions that would otherwise be “illegal.”

The Sobolev Spaces

Now that we can differentiate rough functions (in the weak sense), we need a space to put them. We need a vector space of functions that have “finite energy”, a space where we can measure the size of both a function and its derivatives. This is the natural next step after the minimizer and gradient viewpoint introduced in Part I.

Measurable Functions

Let us take a small digression to measurable functions as it is pretty fundamental to understand what we are doing. As we already discussed we do not measure points because they have 0 measure, so we only measure subsets. We therefore define a family B of subsets of Ω that we can measure. Measurable sets are just any sets which belongs to B. These are the sets we can consistently assign a size to, built from rectangles (products of intervals) using countable unions, intersections, and complements If we have function u:Ω→R, we want to transport this measure to R. To find measure of outcome set A⊂R, we must look at inverse image u−1(A) in Ω. For this to work, u−1(A) must be in B. Since we care about intervals I⊂R, we want u−1(I) to be measurable. This is same as saying u−1([−∞,a))5 is measurable ∀a∈A. So intuitively, measurable function just guarantees inverse image of intervals are measurable, which allows us to transport measure and compute integral.

The Lp Spaces

First, we need a way to measure how big a function is. It is convenient to measure size using integrals rather than maximum values (which are too delicate; changing a function at a single point can ruin its supremum, but doesn’t affect integrals because points have measure zero).

For 1≤p<∞, we define the space Lp(Ω) (where Ω is an open subset in Rn) as the set of measurable functions such that

∥u∥Lp​=(∫Ω​∣u∣pdx)1/p<∞.

(When p=2, this is like an infinite-dimensional version of the Euclidean length ∑ui2​​.)

Remark 1

Strictly speaking, Lp consists of equivalence classes of functions where two functions are considered the same if they differ only on a set of measure zero.

Definition 5 (Sobolev Space)

The Sobolev space W1,p(Ω) contains every function u∈Lp(Ω) whose first weak derivatives ∂x1​∂u​,…,∂xn​∂u​ also belong to Lp(Ω). We measure functions in this space with the norm

∥u∥W1,p​=(∥u∥Lpp​+∑i=1n​​∂xi​∂u​​Lpp​)p1​.

More generally, one can define Wk,p(Ω) by requiring weak derivatives up to order k to be in Lp, but for our purposes k=1 suffices. In-fact, W1,2(Ω) is important to us. It is often written as H1, which is a Hilbert space. The minimizer in Hilbert’s 19th problem we are trying to study initially lives in H1. This space is also complete6, therefore once we prove the sequence of approximate minimizers is a Cauchy sequence, we can be sure that their limit, our desired minimizer, exists in the space itself. A nice thing about Hilbert space is that the norm comes from an inner product:

⟨u,v⟩H1​=∫Ω​uvdx+∫Ω​∇u⋅∇vdx.

Think of it like the dot product between vectors that you already know; we are doing the same thing here. As we are in a vector space and these functions are the elements of that vector space, functions are vectors.

Consider Ω=(0,1)⊂R. Let:

We now verify that both u and w live in H1(0,1)=W1,2(0,1).

For u,

For w,

∥w∥H12​=∫01​w(x)2dx+∫01​w′(x)2dx=∫01​​x−21​​2dx+∫01​(sgn(x−21​))2dx.

The first integral is symmetric about x=21​, so

The second integral is ∫01​(sgn(x−21​))2dx=1. Hence

Both norms are finite, so u,w∈H1(0,1). A natural follow up is: how can we be sure that the minimizers are not rough functions which live outside H1? There exist rough functions that don’t live in H1; for example consider the step function on Ω=(0,1):

For any test function ϕ∈Cc∞​(0,1),

where v(x) is the weak derivative. Thus, the weak derivative in this case is the Dirac delta distribution, h′=δ21​​(x).

In this case, notice that J(u) blows up which is not something we are looking at (if it blows up or is undefined, the problem of finding minimizer itself becomes meaningless). In general, if a rough function is outside the H1 space, then J(u) blows up or is undefined. So to answer that question: the problem (the functional J(u)) itself ensures us that the rough functions which may encounter will certainly live in H1.

Sobolev Inequality

Another natural follow up question is: could an H1 function be really wild? Like, discontinuous everywhere? Unbounded? Oscillating infinitely many times? To answer that we utilize Sobolev inequality.

Definition 6 (Sobolev Conjugate)

For 1≤p<n, the number

p∗=n−pnp​

is called the Sobolev conjugate of p. Notice p∗>p.

Theorem 1 (Sobolev Inequality)

Let Ω⊂Rn be bounded and open. For 1≤p<n, there exists a constant C=C(p,n,Ω) such that for every u∈W1,p(Ω),

∥u∥Lp∗(Ω)​≤C∥u∥W1,p(Ω)​.

To get a sense of this inequality, let us first start with a familiar concept — the Fundamental Theorem of Calculus:

u(x)=u(a)+∫ax​u′(t)dt.

If u′∈Lp(0,1), then from Hölder’s inequality,

∣u(x)−u(a)∣≤​∫ax​u′(t)dt​≤∥u′∥Lp​∣x−a∣1/q,where p1​+q1​=1.

What we have here is, we started with the information u′∈Lp(0,1) and then we ended with the fact that u∈L∞ since u is Hölder continuous. Similarly, Theorem 1 says that if we start with ∇u∈Lp, then we end up with u∈Lp∗ where p∗>p. Take p=2 (our H1 case). Then the Sobolev conjugate is

2∗=n−22n​.

Let’s check what this means in different dimensions: For n=3: 2∗=16​=6. So H1↪L6. So in 3-dimension, we start with ∇u∈L2, then we end up with u∈L6. Similarly in 2-dimensions, H1 functions belong to Lq for every finite q, but not necessarily L∞. To answer our original question: while an H1 function need not be bounded or continuous (especially in dimensions n≥2), the Sobolev inequality prevents it from being arbitrarily wild. It forces the function into a strictly better integrability class Lp∗, which prevents situations like being discontinuous everywhere or oscillating without control.

Reverse Poincaré Inequality

Before we can state the Caccioppoli inequality, we need one more tool. Recall that the Sobolev inequality controls the function by its gradient. The Poincaré inequality is a related classical result in the same spirit:

Theorem 2 (Poincaré Inequality)

There exists a constant C=C(n)>0 such that for every u∈H01​(Br​),

∫Br​​u2dx≤Cr2∫Br​​∣∇u∣2dx.

Intuitively, if u vanishes on the boundary and ∇u is small, then u cannot stray far from zero. The gradient controls the function. Now can we go in the reverse direction? Can we control the gradient by the function? For a generic H1 function this is impossible: adding a large constant to u changes nothing about ∇u, yet makes ∫u2 arbitrarily large. However, for functions satisfying an elliptic equation, the equation prevents this. A solution cannot be large without its gradient also being large somewhere nearby. This is the content of the reverse Poincaré inequality:

Theorem 3 (Reverse Poincaré Inequality)

Let u∈H1(Ω) be a weak solution to an elliptic equation7 in Ω. Then for concentric balls B2r​​⊂Br​⊂Ω,

∫B2r​​​∣∇u∣2dx≤r2C​∫Br​​u2dx,

where C depends only on the ellipticity constants and the dimension n.

The key feature is the ball shrinkage: we can control ∇u on B2r​​ by u on the strictly larger ball Br​, but not on the same ball. This is the unavoidable price of reversing the Poincaré inequality.

Caccioppoli Inequality

We now know that our minimizer lives in H1, and the Sobolev inequality told us it actually lives in a better space than we thought. The Reverse Poincaré inequality (Theorem 3) then gives us control over ∇u from u. But here is the key insight for the bootstrapping argument: we do not want to apply this to u itself. We want to apply it to v=∂k​u, the k-th partial derivative of u. If v itself satisfies an elliptic equation, which it does by differentiating the Euler-Lagrange equation from Part I, then we can apply the Reverse Poincaré inequality to v and gain control over ∇v=∇∂k​u. This then feeds back into the Sobolev inequality, giving better integrability for ∂k​u, and so on. This is what we mean by bootstrapping. The Caccioppoli inequality makes this precise.

Theorem 4 (Caccioppoli Inequality)

Let v be a solution to an elliptic equation of the form

∑i,j=1n​∫Ω​aij​(x)∂i​v∂j​ϕdx=0for all ϕ∈Cc∞​(Ω),

where the coefficients aij​ are bounded and elliptic (meaning ∑aij​ξi​ξj​≥λ∣ξ∣2 for some λ>0). Then for any ball Br​⊂Ω,

∫B2r​​​∣∇v∣2dx≤r2C​∫Br​​v2dx,

where C depends only on the ellipticity constants and dimension.

For us, v=∂k​u, meaning v is the kth component of ∇u. Notice that we are bootstrapping v=∂k​u, not u itself. By improving the integrability of the gradient’s gradient, we are indirectly improving the regularity of u. This is the analytic version of the regularity story that Part I approached geometrically.

Now what Theorem 4 is basically saying is if we know how large v is on a ball Br​, then on a smaller ball B2r​​ we can control its gradient. More precisely, since v∈L2 (because ∇u∈L2), the right-hand side is finite. Thus, the left-hand side is finite, which means ∇v exists as a weak derivative and is in L2.

But the Caccioppoli inequality alone only gives us ∇v∈L2, which we already knew. To really get the bootstrap going, we need something more, a reverse Hölder inequality8 combined with Gehring’s lemma (see the standard Gehring-lemma literature):

(∣B2r​​∣1​∫B2r​​​∣∇v∣p)p1​≤C(∣Br​∣1​∫Br​​∣∇v∣2)21​

with p=2+ϵ for some ϵ>0 and v is as described above in Theorem 4. Now, this says that ∇v is in Lp. Now from Theorem 1, we get v is in Lp∗. As v=∂k​u, so it is pretty clear as how this improvement is for u also all the way till L∞. Once ∇u∈L∞, the gradient is essentially bounded, but not yet continuous.
